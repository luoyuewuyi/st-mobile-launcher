# ST Terminal Manager

一键安装命令：

```bash
curl -fsSL https://raw.githubusercontent.com/luoyuewuyi/st-mobile-launcher/master/termux/bootstrap.sh | bash
```

安装完成后：

- 打开 `Termux` 会自动进入菜单
- 只有你选择 `0. 退出` 才会回到普通终端
- 关闭后重新打开 `Termux`，也会再次自动进入菜单

菜单：

1. 启动酒馆
2. 版本选择
3. 更新酒馆
4. 更新脚本
5. 查看日志
6. 修改端口
0. 退出

说明：

- 酒馆版本最多保留 3 个
- 脚本版本最多保留 3 个
- 更新脚本后会自动切到新脚本
- 查看日志时按任意键返回
